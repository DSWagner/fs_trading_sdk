feat(conviction): replay sparkline + comparison pair + achievements + route ErrorBoundary + unified ShareKit + lavender OG card

Five flagship features land in one commit to push the competition entry
over the finish line, plus a refreshed OG card and 60+ new tests.

Replay Sparkline. The ConsensusDriftSparkline on the Receipt now exposes
a Play/Pause pill (visible only when there are 3+ snapshots). The pill
toggles an isPlaying flag that drives a requestAnimationFrame loop;
on each tick the [0, 1] playProgress slices the snapshot list to
Math.round(points.length * playProgress). Two SVG paths render: a
faded "ghost" trace of the full path, and a darkening "played" trace
that grows during replay. A PlayheadDot interpolates between the last
fully-played snapshot and the next, so the cursor reads as a smooth
playhead. With playProgress === 1 the rendered visual is identical to
the pre-replay implementation, so nothing regresses for non-replay
viewers.

Comparison Pair. components/ComparisonPair.tsx renders the user's
polaroid next to a synthesised "crowd polaroid" on the Receipt. The
crowd polaroid pulls its prediction, spread, and conviction from
summariseConsensus(curve, lowerBound, upperBound) -- a pure helper
that trapezoidal-integrates the useConsensus density to produce three
scalars the Polaroid renderer needs. The crowd polaroid uses a
crowd-prefixed marketId so it gets a separate procedural seed. A
DiffBand below the pair quantifies the gap and grades who landed
closer post-resolution. summariseConsensus returns null for every
degenerate input (null, empty, fewer than 3 points, zero mass, NaN)
so the block hides itself rather than rendering a misleading frame.

Achievements. achievements.ts is a pure-function module that turns
the local rarity ledger into nine bronze / silver / gold badges:
on-the-record, five-call streak, first verdict, contrarian five,
sharp call, first epic, first legendary, first mythic, calibrator.
Monotonicity is a hard invariant (more bets can never unlock fewer
badges). components/AchievementsStrip.tsx renders a horizontal grid
on the Profile between RarityLedger and CalibrationCard. Locked tiles
greyed with progress markers (2 / 5) + locked-state hint via title=;
unlocked tiles in tier colour with the editorial caption surfaced
via title=. Strip is intentionally non-interactive; no modals.

ErrorBoundary at every route. components/ErrorBoundary.tsx is a
class component (required by React 18) installed in App.tsx around
BOTH the main ConvictionShell Routes AND the EmbedPage Routes, keyed
off location.pathname so a crashed boundary resets automatically on
navigation. The default fallback is on-brand (lavender card, mono
eyebrow with route label, Bricolage headline, ember "Try again"
pill, paper "Back to the front" link) and exposes the error message
in a collapsible details element. Crashes log to console.error only,
no exfiltration. storage.ts was also hardened: every localStorage
operation is wrapped in try / catch so quota-exceeded or
private-browsing errors can never blow up the page.

ShareKit. components/ShareKit.tsx exposes a unified share row on the
Receipt page with four actions: Share (Web Share API, file-bearing
on mobile / Chromium, URL-only fallback, Twitter intent as last
resort), Copy link (clipboard.writeText with execCommand fallback +
1.8 s toast), Download PNG (delegates to the existing
downloadPolaroidPng pipeline). Replaces the older standalone
"Download as PNG" button so the page has ONE share affordance.

OG card refresh. public/og-card.svg was redrawn in the lavender +
ember palette to match the live UI. Three rotated polaroids on the
right (epic / legendary / mythic) over the editorial headline. The
previous card was in the older sepia palette and read as a different
product on Twitter previews.
